
# @nielse63/noise
> A simplex 3d noise algorithm

## Installation

Using Yarn
```bash
yarn add --save @nielse63/noise
```

or NPM
```bash
npm install --save @nielse63/noise
```

## Usage

```js
import noise from '@nielse63/noise';

const ratio = noise(x, y, z);
// do something with the ratio...
```
