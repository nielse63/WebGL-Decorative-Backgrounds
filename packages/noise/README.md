
# @nielse63/noise

> Part of the WebGL Decorative Backgrounds sample

## Installation

Using Yarn
```bash
yarn add --save @nielse63/noise
```

or NPM
```bash
npm install --save @nielse63/noise
```

## Usage

```js
import noise from '@nielse63/noise';

const value = 'something';
const output = noise(value);
// do something...
```
